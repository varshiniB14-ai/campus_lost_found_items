# Campus Lost and Found Portal

A web-based platform designed to help students recover lost belongings on campus. This project provides a centralized system to report, search, and manage lost and found items efficiently.

## 🚀 Features
* **Report Lost Items:** Users can submit details about items they've lost.
* **Post Found Items:** A simple interface for finders to list items.
* **Searchable Database:** Easily filter through listings.
* **Modular Setup:** Secure database handling via `db_config.py`.

## 🛠️ Tech Stack
* **Backend:** Python (Flask)
* **Frontend:** HTML/CSS (Templates)
* **Database:** MySQL / SQLite

## ⚙️ How to Run Locally
1. Clone the repository:
   `git clone https://github.com/varshiniB14-ai/campus_lost_found_items.git`
2. Install Flask:
   `pip install flask`
3. Run the app:
   `python app.py`
